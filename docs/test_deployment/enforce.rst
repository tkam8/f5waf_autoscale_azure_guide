セキュリティポリシーをENFORCEに設定
====================================================

#. **Security** > **Application Security** > **Parameters** > **Parameters List** をクリックします。

   |enf_1|
   
#. ワイルドカード（*）パラメータのチェックボックスを入れ、 **Enforce** をクリックします。

   |enf_2|

#. 変更をポリシーに適用するためにApply Policyをクリックします。

   |enf_3|
